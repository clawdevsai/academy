# Agentes de IA com Claude Code — Roteiro Completo

Curso em HTML sobre desenvolvimento de software com agentes de IA usando o **Claude Code**, do básico ao projeto completo. Acompanha uma coleção de prompts prontos para criar agentes especializados.

## Conteúdo

### 📚 Roteiro (`roteiro-agentes-ia/`)

15 capítulos em HTML. Abra `roteiro-agentes-ia/index.html` no navegador para começar.

| Cap | Tema |
|-----|------|
| 01 | Introdução ao desenvolvimento de software com IA |
| 02 | O que é o Claude Code e como funciona |
| 03 | O que são AI Agents e por que utilizá-los |
| 04 | Prompts, Comandos, Agentes, Tools, MCPs, Skills e Hooks |
| 05 | Como estruturar um workflow profissional |
| 06 | Dividindo responsabilidades entre múltiplos agentes |
| 07 | Como criar e organizar agentes personalizados |
| 08 | Spec-Driven Development |
| 09 | Revisão de código, QA, segurança, DevOps e arquitetura |
| 10 | Mantendo contexto entre tarefas e reduzindo tokens |
| 11 | Boas práticas para projetos reais |
| 12 | Erros mais comuns de iniciantes e como evitá-los |
| 13 | Demonstração completa — projeto com múltiplos agentes |
| 14 | Checklist final de boas práticas |
| 15 | Workflows Dinâmicos com Agentes (os 6 padrões) |

### 🤖 Prompts (`prompts/`)

Prompts prontos para criar agentes e hooks:

- `criar-agente-backend-architect.md` — arquiteto de backend
- `criar-agente-backend-enginner.md` — engenheiro de backend
- `criar-agente-code-review.md` — revisor de código
- `criar-agente-cyber-security.md` — segurança
- `criar-agente-devops-engineer.md` — DevOps
- `criar-agente-qa-engineer.md` — QA
- `criar-hook-pre-commit.md` — hook de pre-commit

## Como usar

1. Clone o repositório
2. Abra `roteiro-agentes-ia/index.html` no navegador
3. Siga os capítulos em ordem
4. Use os prompts de `prompts/` no seu próprio Claude Code

## Requisitos

- Navegador web (roteiro é HTML estático, sem build)
- [Claude Code](https://claude.com/claude-code) para praticar os exemplos
